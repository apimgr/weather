Let's set up your Weather Service

First-Time Setup

Welcome to Weather Service! Since this is your first time running the application, we need to set up your accounts.

Setup Process

  1. Create Your Account - You'll create the owner account (regular user)
  2. Create Administrator - Then create a separate administrator account for managing the system
  3. Start Using - Access your weather dashboard and features
Note: The first account you create will be the owner account with regular user privileges. You'll then create a separate administrator account for system management.
